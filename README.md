# 📈 Retail Sales Performance Analytics (Python)
## 🎯 Business Context
The regional sales directors need to evaluate quarterly performance metrics across different product categories to optimize inventory allocation and identify underperforming territories.
## 🛠️ Analytical Approach
Using Python's data analysis libraries, I developed a structured routine to:
- Aggregate raw regional transactional items into clean financial segments.
- Calculate key growth benchmarks (Total Profit Margin, Contribution %).
- Isolate low-performing sectors requiring targeted promotional campaigns.
## 📁 Project Contents
- `sales_analysis.py`: Main data processing script.
